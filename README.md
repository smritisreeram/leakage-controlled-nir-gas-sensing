# Leakage-Controlled Machine Learning for Broadband NIR Gas Sensing

**Working repository** for the machine learning part of the paper.

This repository contains the Jupyter notebooks needed to reproduce the classification and regression experiments described in the paper.

> **Note**: The full paper (including sensor hardware section) is still in preparation. The final title and citation will be updated in a future release.

## Notebooks

- `classification.ipynb`
- `regression_acetone.ipynb`
- `regression_ethanol.ipynb`
- `regression_ammonia.ipynb`
